# 🩺 Public Health Data Analysis – Diabetes Dataset

## 📌 Project Overview
This project focuses on analyzing a diabetes dataset using data preprocessing and Exploratory Data Analysis (EDA) techniques. The goal is to understand patterns, relationships, and key health indicators associated with diabetes.

The project is implemented using Python and visualized through an interactive Streamlit dashboard.

## 🎯 Objectives
- To load and understand the dataset
- To perform Exploratory Data Analysis (EDA)
- To handle missing and incorrect values
- To remove outliers using the IQR method
- To normalize the dataset
- To visualize important health features
- To identify patterns related to diabetes

## ⚙️ Methodology

### 1. Data Collection
- Dataset based on Pima Indians Diabetes Dataset

### 2. Data Preprocessing
- Replaced missing/zero values using median
### 3. Outlier Removal
- Used Interquartile Range (IQR) method

### 4. Data Normalization
- Applied Min-Max scaling (0 to 1 range)

### 5. Exploratory Data Analysis (EDA)
- Statistical summary
- Feature relationship analysis

### 6. Data Visualization
- Histogram (feature distribution)
- Boxplot (outlier detection)
- Countplot (class distribution)
- Heatmap (correlation)
- Scatter plot (feature relationships)
